**📘 异或链表原理简述：**

- 基本原理：
  1. 异或满足交换律与结合律；
  2. `0^x = x`, `x^x = 0`；
  3. 通过存储 `prev^next` 达到节省指针空间目的。

- 缺点：
  - 无法随机访问；
  - 插入删除复杂；
  - 调试困难。
